# 📡 ステルス技術シミュレーター (Stealth Technology Simulator)

TypeScript と HTML5 Canvas で実装された、2Dのステルス性（レーダー反射特性）視覚化シミュレーターです。
航空機の形状（形状制御）、レーダー波の照射角度、および表面素材（電波吸収材: RAM）の相互関係が、レーダー反射断面積（RCS）にどのように影響するかをリアルタイムに学ぶことができます。

## 🚀 主な機能

- **3種類の機体形状シミュレーション**
  - **従来型戦闘機 (デルタ翼):** レーダー波をあらゆる方向に乱反射・帰還させやすい形状。
  - **多面体ステルス (F-117風):** 直線と平らな面で構成され、特定の角度以外では電波を明後日の方向へ逃がす形状。
  - **全翼機 (B-2風):** 翼の前縁・後縁の角度を統一（アライメント）し、反射方向を極限まで限定した形状。
- **レーダー照射角の動的変更:** スライダーでレーダーの照射角度を変え、どの角度で「強い反射（探知）」が起きるかを可視化。
- **電波吸収材（RAM）調整:** 表面素材の電波吸収率を変えることで、反射波全体のエネルギー減衰をシミュレート。
- **リアルタイムRCS（レーダー反射断面積）計算:** 帰還した電波の総量から、見かけの大きさを $㎡$ 単位で擬似計算して表示。

📐 シミュレーションの仕組み（技術的背景）レイトレーシング（Ray Tracing）: レーダー発信源から並行な電波（ベクトル）を照射し、機体の各セグメント（線分）との交点を判定します。鏡面反射（Specular Reflection）: 交点における面法線ベクトル（Normal）を算出し、入射角に応じた反射ベクトルを $I - 2(I \cdot N)N$ で計算します。探知判定: 反射したレイがレーダー発信源の方向へ戻っているかを内積（Dot Product）を用いて判定。ほぼ逆向きに戻っている場合、警告（赤いライン）として強調され、RCSの値を上昇させます。エネルギー減衰: 設定されたRAM（電波吸収材）の効率に応じて、反射波の不透明度（アルファ値）およびRCSへの影響度を減衰させます。📝 ライセンスMIT License
